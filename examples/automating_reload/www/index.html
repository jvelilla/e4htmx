Automating Reload (EWF WebSockets)

This example demonstrates the client-side code needed for automating browser reloads via WebSockets, completely integrated into the Eiffel Web Framework server!

To test this, try editing this `index.html` file in your editor while the server is running. As soon as you save, the WebSocket server will detect the file change and command this tab to reload!
