Glimmer Templates: Filters & Helpers

Welcome to the interactive showcase for Glimmer's variable transformation pipeline. Learn how to format data in your views using built-in filters, chained operations, and custom Eiffel agents.

Built-in Filters

Text Case

Built-in

Transforms strings to uppercase or lowercase directly in the template.

Syntax: {name | upper}
{user_name} → {user_name | upper}
Syntax: {name | lower}
{user_name} → {user_name | lower}

Text Truncating

Built-in

Shortens strings to a maximum length without breaking words, followed by an optional arg.

Syntax: {text | truncate: 45}
{description | truncate: 45}

Date & Time

Built-in

Formats Date, DateTime, or epoch timestamps using format string tokens.

Syntax: {date | date_format: "dd/MM/yyyy"}
{raw:created_at_raw} → {created_at | date_format: "dd/MM/yyyy"}
Syntax: {date | date_format: "yyyy-MM-dd HH:mm"}
{created_at | date_format: "yyyy-MM-dd HH:mm"}

Decimals

Built-in

Formats numeric variables to a fixed number of decimal places with rounding.

Syntax: {value | number_format: 2}
{score} → {score | number_format: 2}

Currency Formatter

Built-in

Renders float or integer values with ISO currency symbols prefix or suffix.

Syntax: {amount | currency: "USD"}
{balance | currency: "USD"}
Syntax: {amount | currency: "EUR"}
{balance | currency: "EUR"}

Filter Chaining

Combined

Sequence transformations together. Output flows from left to right through the pipeline.

Syntax: {name | lower | truncate: 4}
"John Doe" → {user_name | lower | truncate: 4}

Custom Eiffel Helpers

Robot Avatar

Eiffel Agent

Calls an Eiffel agent function mapping the email to a Dicebear robot SVG URL.

Syntax: <img src="{email_addr | gravatar_url}" />
{email}

HTML Status Badge

Eiffel Agent

Converts raw strings into beautifully styled status pill HTML components (using raw rendering).

Syntax: {raw:status_val | status_badge}
"{status}" → {raw:status | status_badge}

Slugify Text

Eiffel Agent

Converts regular titles and sentences into URL-friendly lowercase-hyphenated slugs.

Syntax: {text | slugify}
"My Page" → {app_title | slugify}
